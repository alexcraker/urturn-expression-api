This page describe the various dialog that can be called from a Post#dialog(type, options, callback)

image

.dialog('image', options, callback);

  • options: 
    {
  size : // crop options
  { 
    width : 23px, // asked width
    height : 23px, // asked height
    adaptUI : true, // true mean we should adapt camera screen UI to this size
    auto : true, // Mean it will autoCrop to this size
    flexRatio : flase // true : when crop panel is display, user can change ratio of crop zone
  },
  applyShaders : 
  [
    ..., // a json array discribing filters. See [Filter documentation]()
  ],
  inputType : [  // if specified only first types list here will be available
    'camera', // the camera
    'search', // the search
    'social', // the link input
    'myUpload' // user uploads
  ]
}

  • callback(String) - will be passed the resulting string or null if no value have been selected.

text

.dialog('text', options, callback)

  • options: 
    {
  value: String      // the default value
  max: Integer       // the maximum number of chars that might be entered
  multiline: boolean // if true, the text input will be multiline
}
  • callback(String) - will be passed the resulting string or null if no value have been selected.

crop

.dialog('crop', options, callback)

Let you show a crop dialog to crop an image

  • options: 
    {
  image : UT.Image Object,
  size {
    width : 100px,
    height : 100px,
    flexRatio : true
  }
}
  • callback(String) - will be passed the resulting image.
