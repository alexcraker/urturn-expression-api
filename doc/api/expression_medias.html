UT.Expression#medias instance

expression.medias.imageDialog

expression.medias.crop(image, options, callback)

Parameters

image : An image object return by imageDialog or an url to an image.
options : A hash of options :

  • width : width of croped image height : height of croped image auto : [true | false], define if image need to be croped automaticly or if a cropDialog need to be showed to user

callback : A callback called when image has been croped (with new image in parameters)

expression.medias.createImage

Parameters :
- urlOrBase64 : url of an image of dataUrl of an image
- callback : callback called when image is created.
Usage : 

var url = document.getElementById('canvas').toDataURL();

  expression.medias.createImage(
    url,
    function (data, error) {
      if (error) {
        console.error('Image Creation failer : ' + error);
      }
      else{
        console .log('Image creation succed : ' + data);
      }
    });
})
Usage with items.save : 
var url = document.getElementById('drawZone').toDataURL();

  expression.medias.createImage(
    url,
    function (data, error) {
      expression.items.save('image', data, function () {
        console.log('Image Created and saved');
      }); 
    });
})
