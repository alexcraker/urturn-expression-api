UT.Expression

UT.Expression instances contains all the methods and informations needed to create, view and publish Urturn posts.

Public Class Methods

UT.Expression.ready( callback )

Register a callback to be called once the expression is ready to be initialized. An instance of UT.Expression is passed to the callback.

Callback Arguments

expression
UT.Expression instance to use in this expression.

Example

<script type="text/javascript">
UT.Expression.ready(function(expression){
  new MyClass(expression);
});
</script>

Public Instance Methods

UT.Expression#post( callback )

Register a callback to be called when the user asked for the expression to be posted. The callback must call the given doPostCallback in return.

Note that you don't need to wait on UT.Expression#items.save() callback as we do that for you during the post publication operation.

Callback Arguments

doPostCallback
function that must be called in any case once all post operations are done.

UT.Expression#getNote()

Retrieve the value of the post note.

Return value

String: the value of the post note.

Usage


  UT.Expression.ready( function( expression ){
    var note = expression.getNote();
  });

UT.Expression#setNote(Value)

Set the value of the post note.

Parameters

  • note[String]: url of an image or dataUrl of an image

Returned value

undefined

Usage


  UT.Expression.ready( function( expression ){
    var note = expression.setNote("Guess what I'm cool #guesscool");
  });
