Urturn API support the following filters :

  • Brightness
  • Contrast
  • Hue
  • Saturation
  • Sepia
  • Threshold
  • Color Invert
  • Vignette
  • Gamma
  • Exposure
  • Color
  • Noise

Brightness Filter

The Brightness filter let you change image Brightness.

Parameters :

  • strenght : from -1.0 to 1.0, default 0.0 Let you control brightness strenght. -1.0 is for a black image, 1.0 is for a white image, 0.0 leave image as it is.

example: json { "filter" : "brightness", "parameters" : { "strength" : 0.2 } }

Contrast

The Contrast filter let you change image Contrast.

Parameters :

  • strenght : from -1.0 to 1.0, default 0.0 Let you control contrast strenght. -1.0 is for a grey image, 1.0 is for a high contrast, 0.0 leave image as it is.

example: json { "filter" : "contrast", "parameters" : { "strength" : 0.2 } }

HUE

The HUE filter let you change image HUE.

Parameters :

  • strenght : from -1.0 to 1.0, default 0.0 Let you control hue strenght. 0.0 leave image as it is.

Saturation

The Saturation filter let you change image saturation.

Parameters :

  • strenght : from -1.0 to 1.0, default 0.0 Let you control saturation strenght. 0.0 leave image as it is. -1.0 give you a desaturated (black and white) image, 1.0 an over saturated one.

example: json { "filter" : "saturation", "parameters" : { "strength" : 0.2 } }

Sepia

The Sepia filter let you apply a yellow-black filter on your image.

Parameters :

  • strenght : from 0.0 to 1.0, default 0.0

    Let you control sepia strenght. 0.0 leave image as it is. 1.0 apply a full sepia filter

example: json { "filter" : "sepia", "parameters" : { "strength" : 0.2 } }

Threshold

The Threshold filter transform your image to a 2 different color image.

Parameters :

  • strenght : from 0.0 to 1.0, default 0.5

    Let you control the threshold limit. 0.5 mean all color under 128 will be of primary, all color up of secondary.

  • alpha : from 0.0 to 1.0, default 1.0

    Let you control the alpha of this filter.

  • primaryColor : Array of 3 number defining RGB chanels [r (0-255),g (0-255),b (0-255)], default [0,0,0];

    Let you define the primary color.

  • secondaryColor : Array of number defining RGB chanels[r (0-255),g (0-255),b (0-255)], default [255,255,255];

    Let you define the secondary color.

example: json { "filter" : "threshold", "parameters" : { "strength" : 0.2, "alpha" : 0.5, "primaryColor" : [24,44,33], "secondaryColor" : [223,222,123] } }

Color invert

Invert color on your image.

Parameters :

  • No Parameters !

example: json { "filter" : "invertColor" }

Vignette

Add a Vignette in top of your image

Parameters :

  • amount : from 0.0 to 1.0, default 0.0 The amount of vignette added to your image, 0.0 no vignette, 1.0 max amount

  • size : from : 0.0 to 1.0 The size of the vignette border : 0.0 no vignette, 1.0 max size

  • centerX : from 0.0 to 1.0, default 0.5 The X coordinate of center of your vignette. 0.0 mean totaly left, 1.0 totaly right of your image. 0.5 mean exact middle of your image.

  • centerY : from 0.0 to 1.0, default 0.5 The Y coordinate of center of your vignette. 0.0 mean totaly top, 1.0 totaly bottom of your image. 0.5 mean exact middle of your image.

  • r : from 0 to 255, default 0 The red chanel of vignette color

*g : from 0 to 255, default 0 The green chanel of vignette color

*b : from 0 to 255, default 0 The blue chanel of vignette color

example: json { "filter" : "vignette", "parameters" : [ "size" : 0.2, "amount" : 0.3, "centerX" : 0.5, "centerY" : 0.5, "r" : 12, "g" : 234, "b" : 123 ] }

Gamma

The Gamma filter let you change gamma of your image.

Parameters :

  • strenght : from 0.0 to 3.0, default 1.0

    Let you control gamma strenght. 0.0 image is white. 1.0 image intact. 3.0 Strong Gamma

example: json { "filter" : "gamma", "parameters" : { "strength" : 0.2 } }

Exposure

The Exposure filter let you change exposure of your image.

Parameters :

  • strenght : from 0.0 to 1.0, default 0.0

    Let you control exposure strenght. 0.0 image is untouched. 1.0 image over-exposed.

example: json { "filter" : "exposure", "parameters" : { "strength" : 0.2 } }

Color

The color filter let you change colo balance of your image.

Parameters :

  • a : from 0.0 to 1.0, default 0.0

    Let you control alpha of this filter

  • r : from 0.0 to 1.0, default : 0.0

    Let you control influence of this filter to red chanel.

  • g : from 0.0 to 1.0, default : 0.0

    Let you control influence of this filter to green chanel.

  • g : from 0.0 to 1.0, default : 0.0

    Let you control influence of this filter to blue chanel.

example: json { "filter" : "color", "parameters" : { "a" : 0.2, "r" : 0.3, "g" : 0.7, "b" : 0.4 } }

Noise

Add noise to your image.

Parameters :

  • Amount : from 0.0 to 1.0, default 0.0

    Let you control amount of noise on your image. 0.0 image is untouched. 1.0 lot of noise.

example: json { "filter" : "noise", "parameters" : { "amount" : 0.2 } }
