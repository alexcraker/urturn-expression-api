Work with Sound

This guide describe how to insert, create or save sound in your expression.

All the function related to sound insertion are in UT.Expression instance #medias namespace.

Insert a sound into your expression

The easiest way to insert an image in your expression is by asking one to Urturn. Use the UT.Expression#medias.dialog method for this.

UT.Expression.ready( function( expression ) {
  expression.medias.dialog('sound', {inputTypes : ['take_media']}, function( soundResource ) {
    expression.storage.myVoice = soundResource;
    expression.storage.save();
  } ) ;
} ) ;
