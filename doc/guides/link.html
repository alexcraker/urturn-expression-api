Create links (no kidding)

To redirect user to an other webpage it's common to use window.location in Javascript. Unfortunately if you do so this will not work well inside our mobile app. It will also require you to guess urturn.com urls if you want to link to a page inside urturn.com.

The API provide an easy way to do so without having to implement 'hacks' via the .navigate method.

Example 1 : redirect user to an external website

UT.Expression.ready(function(post) {
  // Create your post using the post API
  post.navigate('http://example.org');
})

You can also redirect user to search and profil page without having to guess the correct url. This way you will not have to update your expression the day we change our url scheme. It will also work nicely inside our mobile app.

Example 2 : redirect user to a urturn.com search page (when he click an hash tag for example)

UT.Expression.ready(function(post) {
  // Create your post using the post API
  post.navigate('search', 'this is a search query');
})

Example 3 : redirect user to a urturn.com profile Page

UT.Expression.ready(function(post) {
  // Create your post using the post API
  post.navigate('user', 'anUserName');
})
