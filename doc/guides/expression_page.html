Expression Page

Your expression as a dedicated page at URL: http://www.urturn.com/username/expression_system_name.

By default this page looks a little bit empty so here is how to improve it.

Give a title

Your expression will look great with a title. You just have to define it in your expression.json file. 

{
  "system_name":"monkey_paradise",
  "version":"1.0.0",
  "title":"Great Monkey Live in Paradise"
}

Setup the expression banner

The banner will be displayed on the top of the expression page. To define a banner, add a file 'resources/banner.(png|gig|jpg)' in your expression.

The ideal size of the banner is 500x255 pixels and is always shown with a ratio of this if smaller. Never shown bigger nor stretched.

Setup the expression icon

To define an icon, add a file 'resources/icon.(png|gig|jpg)' in your expression.
