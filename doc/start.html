From undefined to "Hello World" in 5 minutes

Create and deploy your first expression

Install the urturn toolbelt

To work and deploy an expression, you will need some precious tools. All of them are available in our command line, working on top of NodeJS 0.8.x (not compatible with NodeJS 0.10.0 at this time). 
Prerequisite: install NodeJS
npm install urturn-toolbelt -g

Prerequisites

  • >= Ruby 1.8.7
  • An Urturn Account

Compile and Install the Urturn Command Line Tool

The command line tool will let you:

  • generate an expression typical file structure
  • publish it
  • manage it
gem install urturn-cli

Create a new expression

Go into a folder which will be the parent folder of your expression project and create the expression:

cd ~/Code
urturn expression:new hello_world
cd hello_world
ls

This will generate the following tree structure

hello_world/                      # expression home
hello_world/editor.html            # editor template file
hello_world/player.html            # player template file
hello_world/expression.json        # expression descriptor file
hello_world/hello_world.css        # expression styles
hello_world/hello_world.js         # expression javascript file
hello_world/hello_world_editor.css # expression styles for editor
hello_world/hello_world_editor.js  # expression javascript for editor

editor.html and player.html will be combined with your script and stylesheet to become your expression template.

Deploy the expression

You can deploy your newly created expression:

urturn expression:deploy

Once logged in with your email and password and the expression uploaded, point your browser to the outputted URL (http://urturn.com/your_username/hello_world)
