expression.url

Introduction

The URL api is a set of helper to help you work with ressources URLs inside an expression.
The points it try to solve are :

  • Allow you to use ressources from other domains inside an expression (proxify, getAssetThroughProxy)
  • Allow you to easily access the ressources you bundle with your expression (getAssetPath,for, getAssetThroughProxy)


expression.url.proxify(url)

Parameters

url : The url of the ressource you want to proxify

Return value

String : The url that will let you access the wanted ressource through a proxy.

Usage

The proxify method will let you retrieve an ressource throught a proxy so you can use it without fear of cross domain issues in you expression. 


  UT.Expression.ready( function( expression ) 
  {
    var ressourceURL = 'http://exemple.org/test.png';
    var proxifyURL = expression.url.proxify(ressourceURL);

    console.log(proxifyURL);
    // Should show something like : urturn.com/proxy/exemple.org/test.png
  });


expression.url.getAssetPath(relativeAssetPath)

Parameters

relativeAssetPath : The relative url of the ressource you want to access. (should be in your expression bundle)

Return value

String : The url that will let you access the wanted ressource.

Usage

The getAssetPath method will let you retrieve an ressource inside your expression bundle using it's relative path. 


  UT.Expression.ready( function( expression ) 
  {
    var assetURL = './img/superMan.gif';
    var ressourceURL = expression.url.proxify(assetURL);

    console.log(ressourceURL);
    // Should show something like : apps.urturn.com/expression/test/1.2.3/img/superMan.gif
  });


expression.url.for(relativeAssetPath)

Parameters

relativeAssetPath : The relative url of the ressource you want to access. (should be in your expression bundle)

Return value

String : The url that will let you access the wanted ressource.

Usage

for is an alias of getAssetPath


expression.url.getAssetThroughProxy(relativeAssetPath)

Parameters

relativeAssetPath : The relative url of the ressource you want to access. (should be in your expression bundle)

Return value

String : The url that will let you access the wanted ressource through a proxy.

Usage

The getAssetThroughProxy method will let you retrieve an ressource from your expression bundle through a proxy using it's relative path.
This is usefull to prevent cross domain security error when you work with your ressources. 


  UT.Expression.ready( function( expression ) 
  {
    var assetURL = './img/superMan.gif';
    var ressourceURL = expression.url.getAssetThroughProxy(assetURL);

    console.log(ressourceURL);
    // Should show something like : urturn.com/proxy/apps.urturn.com/expression/test/1.2.3/img/superMan.gif
  });
