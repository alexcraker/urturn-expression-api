How to use filter

All the function related to storage are in your UT.Expression instance #storage and #items namespace.

The Simple storage API

UrTurn provide 2 way to store datas. The simpe API wich is synchronous for the expression developper and the item API wich is asynchronous.

How to use the simple storage API?

To use the simple storage API, you need to use the expression.storage object. Every properties of this object will be saved and can be retrieve in other instance of the current posts.
This API try to be similar to W#C localStorage API. You can consider the expression.storage object as persistant accros all instances of a POST in view and edit mode. 


  UT.Expression.ready( function( expression ) 
  {
    if(expression.storage.hello){
      console.log(expression.storage.hello);
    }
    expression.storage.hello = 'hello WoRLD';
    // TMP : To be sure datas are saved
    expression.storage.commit();
  });


You can also use the storage object as an Array : 

  UT.Expression.ready( function( expression ) 
  {
    if(expression.storage['hello']){
      console.log(expression.storage['hello']);
    }
    expression.storage['hello'] = 'hello WoRLD';
    // TMP : To be sure datas are saved
    expression.storage.commit();
  });


You can also use the storage object using getItem and setItem methods : 

  UT.Expression.ready( function( expression ) 
  {
    if(expression.storage.getItem('hello')){
      console.log(expression.storage.getItem('hello');
    }
    expression.storage.setItem('hello', 'hello WoRLD');
    // TMP : To be sure datas are saved
    expression.storage.commit();
  });

storage.getItem(key, [value])

Allow you to retrieve a stored information in key key.
Support an optional value parameter. If no item with is found with key : key, value is returned;

storage.setItem(key, value)

Allow you save an information of value : value, under the key : key
