Urturn Command Line Tools

The command line let you manage an expression and its related documents.

Create an expression

urturn expression:new NAME
$ urturn expression:new hello_world

Deploy an expression

urturn expression:deploy [PATH]
$ urturn expression:deploy
$ urturn expression:deploy /Users/john/Code/hello_world

Retrieve a list of all my expressions

 $ urturn expression:list

Retrieve a list of all version of an expression

urturn expression:show NAME
$ urturn expression:show hello_world

Migrate all document using an expression from old version to new version

urturn expression:migrate NAME 'OLD_VERSION' 'NEW_VERSION'
$ urturn expression:migrate hello_world '1.0.0' '1.0.1'
$ urturn expression:migrate hello_world '~1' '1.3.8'
